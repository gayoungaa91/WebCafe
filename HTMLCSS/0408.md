### 웹관련 용어 css
```
.term{
    margin-top:20px;
    background:#ccc linear-gradient(#ccc, #eee);
    border:1px solid #aaa;
    border-radius: 5px;
    padding:10px 15px;
    /* 그라디언트 지원 안하는 브라우저를 위해서 맨 앞에 배경 색깔을 지정해준다. */
}
.term-heading{
    font-size:1.5rem;
    font-weight: 700;
}

.term div{
    margin-top:10px;
}
.term-list-subject{
    margin-top:-4px;
}
.term-list-subject, .term-list-brief{
    float: right;
    width:145px;
}
.term-list-thumbnail{
    float: left;
}
.term-list-thumbnail img{
    vertical-align: top;
}

.term-list-brief{
    margin-top:5px;
}
.even .term-list-subject, .even .term-list-brief{
    float: left;
}
.even .term-list-thumbnail{
    float: right;
}
```
